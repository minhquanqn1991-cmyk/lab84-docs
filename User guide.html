1. Giới thiệuOverview

Lab84 là bộ addin Revit do Hiển Phan phát triển, tích hợp nhiều nhóm công cụ chuyên biệt vào một tab ribbon duy nhất. Tool hỗ trợ Revit 2022 đến 2027.

Lab84 được xây dựng để bù đắp những chỗ Revit gốc còn thiếu hoặc làm chậm khi làm việc theo nhóm và xử lý dữ liệu quy mô lớn.

Lab84 is a Revit add-in developed by Hiển Phan, consolidating multiple specialized tool groups into a single ribbon tab. It supports Revit 2022 through 2027.

Lab84 fills the gaps where vanilla Revit is slow or absent — especially for team workflows and large-scale data management.

3. License & Trial 30 ngày& 30-day trial

Lần đầu cài, addin tự kích hoạt Trial 30 ngày — dùng đầy đủ chức năng, không giới hạn. Sau 30 ngày, addin vẫn load nhưng các tool nghiệp vụ bị khoá (panel Admin vẫn hoạt động để bạn nạp license).

On first install, the add-in starts a 30-day trial — full features, no limits. After 30 days, the add-in still loads but business tools are locked (the Admin panel still works so you can install a license).

3.1 Kích hoạt licenseActivate a license

  1. Vào tab Lab84 → panel Admin → nút License. Open the Lab84 tab → Admin panel → click License.
  2. Bấm Copy Machine ID → gửi mã này cho nhà cung cấp qua email. Click Copy Machine ID → send the ID to the vendor by email.
  3. Nhà cung cấp gửi lại file .lic. The vendor returns a signed .lic file.
  4. Trong dialog License, bấm Nạp file .lic → chọn file vừa nhận → Activated. In the License dialog, click Install .lic file → pick the file → Activated state.

Thêm hoặc xoá Revision trên nhiều Sheet hàng loạt. Adds or removes a Revision from many Sheets in batch.

Bố cục 2 cột2-column layout

  • Cột trái — Revision: Danh sách Revision trong project để chọn Revision cần xử lý.Left column — Revision: List of Revisions in the project — select which Revision to process.
  • Cột phải — Sheet: Danh sách Sheet để tick những sheet cần gán hoặc gỡ revision; hỗ trợ chọn nhanh theo Sheet Schedule có sẵn.Right column — Sheet: List of Sheets — tick the ones that should have or lose the revision; supports quick selection from an existing Sheet Schedule.

Cơ chế Apply: đây là thao tác đồng bộ 2 chiều, không phải 2 nút Add/Remove riêng — tick các Revision cần xử lý ở cột trái, tick các Sheet cần có revision đó ở cột phải, rồi bấm Apply: sheet đã tick sẽ được thêm revision, sheet chưa tick sẽ bị gỡ revision đó (nếu đang có) — trong cùng 1 lần bấm. Apply mechanism: this is a 2-way sync, not two separate Add/Remove buttons — tick the Revisions to process on the left, tick the Sheets that should have that revision on the right, then click Apply: ticked sheets get the revision added, unticked sheets get it removed (if present) — all in one click.
